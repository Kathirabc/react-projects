# 🧠 React Quiz App

A simple and interactive **Quiz Application** built using **React.js**, **React Router DOM**, **Context API**, and **Vite**.  
Users can enter their name, attempt multiple-choice questions, navigate through the quiz, and view their final score on a result page.

## 🧠 How It Works

1. User enters their name on the start screen
2. Quiz begins with multiple-choice questions
3. User selects an answer and clicks **Next**
4. Context API stores score and progress
5. After last question, user clicks **Finish**
6. Final score is displayed on the Result page

---

## 📊 State Management (Context API)

* Stores user name
* Tracks current question index
* Stores selected answers
* Calculates final score
